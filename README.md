# Register Form - Simple Competition Project 

## About
This project was built for a competition
The code is written in a clean and organized way✨(Clean Code)


## Validation Rules
- Both fields empty → error message + red highlight
- Username filled, password empty → fill password message
- Password filled, username empty → fill username message
- Password contains letters → must be numbers only
- Wrong credentials → incorrect message

## Tech Stack
- C# Windows Forms
- NET Framework
- Visual Studio
